ghi nhớ: Khi nào thì nên đặt input và button trong form? -Khi bạn muốn thu thập dữ liệu người dùng và gửi đi (qua JavaScript hoặc HTTP). - Khi bạn dùng nút button type="submit" để gửi form. Khi nào không cần đặt trong form? - Nếu bạn chỉ dùng button để gọi JavaScript và không gửi dữ liệu gì. - Nếu bạn dùng input chỉ để hiển thị, lọc tạm thời, hoặc làm chức năng nội bộ (ví dụ lọc danh sách theo từ khóa).
